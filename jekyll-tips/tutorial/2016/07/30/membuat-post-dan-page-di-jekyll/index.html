Struktur Folder

<source>
 ├─── _includes
 ├─── _layouts
 |    ├── post.html
 |    └── page.html
 ├─── _post
 |    └── 2016-07-30-nama-post.markdown
 ├─── _sass
 ├─── _site
 ├─── _templates
 ├─── about
 |    └── index.html 
 ├─── _config.yml
 ├─── about.html
 └─── index.html

Cara Biasa / default

Cara ini di lakukan tanpa tambahan plugin apapun hanya menggunakan cara yang di berikan oleh si pembuat jekyll.

Membuat Post

  1. Untuk membuat post masuklah ke direktori (folder) _post, direktori di mana post di lahirkan. Kita hanya bisa membuat post di dalam folder ini, kecuali di rubah setinganya di _config.yml.
  2. selanjutnya buatlah file dengan format nama: [tahun]-[bulan]-[tanggal]-[nama-post].[ekstensi] . eg: 2016-07-30-nama-post.markdown nama-post harus lowercase atau huruf kecil semua, jenis file untuk post yang bisa di gunakan markdown, html, textile.
  3. open file di text editor kesukaan anda. tambahkan yaml front mater di bagian paling atas. Untuk format date / tanggal tahun-bulan-tanggalTJam:menit:detik+(zona waktu ex: gmt+07 / +07:00 untuk Jakarta Indonesia).
  4. Isi post anda di letakan setelah front mater menggunakan markdown / html / textile.
---
layout: post
title: "Nama Post"
date: 2016-07-30T17:28:34+07:00
---

Membuat Page

  1. Ada dua jenis page yang bisa di gunakan, semisal di sini saya akan membuat page about:
    • jenis pertama <source>/about.html
      • cara mengakses secara offline 127.0.0.1:4000/about.html
    • jenis yang kedua <source>/about/index.html
      • cara mengakses secara offline 127.0.0.1:4000/about/index.html
  2. Untuk membuat page jenis pertama yaitu buat file di <source> direktori, dengan nama about.html / about.md, page ini bisa di akses secara default di 127.0.0.1:4000/about.html
Namun semisal kita ingin mengganti page jenis pertama ini menjadi jenis yang kedua tinggal menambahkan permalink di front mater: permalink: /about/ 
---
layout: page
title: About
permalink: /about/ #tambahkan jika ingin mengganti page menjadi jenis ke 2
---

isi page
  1. Membuat page jenis ke dua
    • Cara mengakses page secara default 127.0.0.1:4000/about/index.html
    • Buatlah folder terlebih dahulu dengan nama sesuai page yang ingin di buat di dalam direktori ,
    • Selanjutnya buat file dengan nama index.html / index.md.
---
layout: page
title: About
---

Isi page

Cara Cepat dan Mudah

Cara ini sangatlah mudah karena menggunakan plugin octopress, sehingga menghemat waktu tenaga dan pikiran. Cara ini bisa di gunakan oleh Windows users.

Instalasi Octopress

Cara yang di rekomendasikan oleh para ahli.

  1. Buka terminal atau cmd.
  2. Masuk ke direktori tujuan (letak blog/website) eg: $:~/Desktop/mySite/.
  3. Ketikan perintah bundle init ~> enter, perintah ini akan menghasilkan Gemfile
  4. Tambahkan gem 'octopress' seperti di bawah ini di dalam Gemfile.
  5. ketikan bundle install -> enter.
group :jekyll_plugins do
 gem 'octopress'
end

cara ke 2

  1. Buka terminal atau cmd.
  2. Ketikan perintah gem install octopress untuk menginstal.
  3. Hanya menunggu beberapa detik octopress sudah berhasil di install, yeah.

Membuat Post

  1. Buka terminal dengan ctrl + alt + t(atau cara lainya), atau cmd untuk Windows users dengan win + r ketikan cmd -> enter (atau dengan cara lainya).
  2. Masuk ke direktori tujuan (letak blog/website) eg: $:~/Desktop/mySite/.
  3. ketikan perintah octopress new post "Nama Post", dengan perintah tersebut post sudah berhasil di buat di folder _post
---
layout: post
title: "Nama Post"
date: 2016-07-30T17:28:34+07:00
---

Isi post di sini

Membuat Page

  1. Buka terminal dengan ctrl + alt + t(atau cara lainya), atau cmd untuk Windows users dengan win + r ketikan cmd -> enter (atau dengan cara lainya).
  2. Masuk ke direktori tujuan (letak blog/website) eg: $:~/Desktop/mySite/.
  3. Membuat page
    • Page jenis 1 Ketikan perintah octopress new page "Nama-page" ~> enter. eg: octopress new page "About". Dengan perintah ini akan menghasilkan file about.html, cara mengaksesnya 127.0.0.1:4000/about.html.
    • Page jenis 2 Ketikan perintah octopress new page "Nama-page/index.html" (index.md) ~> enter. eg: octopress new page "About/index.html". Dengan perintah ini akan menghasilkan file about.html, cara mengaksesnya 127.0.0.1:4000/about/index.html.
  4. Untuk mengganti page jenis 1 menjadi jenis ke 2 tambahkan permalink di front mater permalink: /about/, dan cara mengaksesnya secara default yaitu 127.0.0.1:4000/about/index.html
---
layout: page
title: About
---
