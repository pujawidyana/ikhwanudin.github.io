Cara Membuat

Hal pertama yang harus di lakukan yaitu memanggil semua source:

  • bootstrap.min.css diletakan di dalam head,
  • jquery-2.2.2.min.js, bootstrap.min.js, dan jekyll-search.min.js di letakan sebelum penutup tag </body> secara urut.

Button Dropdown

Buat input dengan type="text" yang di adaptasi dari Bootstrap button dropdown seperti di bawah ini:

  <div class="dropdown">
    <!-- Button 1 -->
    <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenu2" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
      Button Dropdown
    </button>
    <!-- Button 2 -->
    <div class="dropdown-menu" aria-labelledby="dropdownMenu2">
      <button class="dropdown-item" type="button">Hasil Pencarian</button>
    </div>
  </div>

Input

  1. Ganti <button>1 dengan input type="text"
    • selanjutnya tambahkan class form-control dan dropdown-toggle,
    • tambahkan atribut data-toggle="dropdown" dan id="hasil".
  2. Ganti <button>2 dengan tag <div>
    • tambahkan id="hasil" dan class="dropdown-menu" seperti code di bawah ini:.
<div class="dropdown">
  <input type="text" id="input" class="form-control dropdown-toggle" data-toggle="dropdown" placeholder="Search...">
  <div id="hasil" class="dropdown-menu"></div>
</div>

Custom css

Tambahkan sedikit css untuk menghilangkan tampilan yang kurang enak di lihat karena pengaruh class="dropdown-menu" di bagian bawah <input>. Untuk mengetahui perubahan tampilan yang terjadi, lihatlah perbedaan tampilan sebelum css di tambahkan dan perbedaan setelah css di tambahkan.

#hasil{
  width: 100%;
}
#hasil.dropdown-menu{
  box-shadow: 0 1px 2px rgba(0, 0, 0, 0.05), 0 5px 10px rgba(0, 0, 0, 0.05);
  padding: unset;
  border: unset;
}

search.json

Buatlah file bernama search.json atau dengan nama sesuai keinginan anda, untuk di generate oleh Jekyll, yang akan meloop seluruh post / page yang ada. Hasil generate file inilah yang akan di tampilkan di bagian id="hasil" ketika user menginputkan value jika ada yang sesuai dengan hasil generate.

  • Letakan file ini di direktori <source>/search.json atau di direktori kesukaan anda.
  • File ini hanya akan megenerate title dan url dari nama post/page.
  • perlu di ingat tambahkan triple dash --- di bagian paling atas file supaya Jekyll megenerate file search.json.

---

---

[
{% assign sorted_posts = site.posts | sort: 'title' %}
{% for post in sorted_posts %}
    {% if post.title %}
        {
            "title" : {{ post.title | jsonify }},
            "url"   : "{{ site.baseurl }}{{ page.url }}"
        } {% unless forloop.last %},{% endunless %}
    {% endif %}
{% endfor %}
]

javascript

Letakan code di bawah ini sebelum tag <body>. Code ini berfungsi untuk memanggil data dari search.json, untuk di tampilkan di dalam tag yang memiliki id="hasil".


SimpleJekyllSearch.init({
  searchInput: document.getElementById('input'),
  resultsContainer: document.getElementById('hasil'),
  searchResultTemplate: '<a class="dropdown-item" href="{url}">{title}</a>',
  noResultsText: '<div class="dropdown-item no-results">Maaf, tidak ada hasil yang sesuai dengan kata kunci.</div>',
  json: '{{ site.baseurl }}/search.json'
})
