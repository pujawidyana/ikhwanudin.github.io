Inti Cerita

  1. Buka Terminal dengan ctrl+alt+t atau Cmd dengan Win+r dan ketikan cmd langsung enter.
  2. Tentukan direktori tujuan yang ingin anda gunakan untuk tempat pembuatan blog dengan Jekyll. Semisal di sini saya menggunakan direktori Desktop/ untuk tempat pembuatan blog / website.
    • Ubuntu :
      • :~$ cd Desktop/ enter;
    • Windows
      • C:/Windows/ cd C:
      • C: cd Users/pengguna/Desktop
  3. Perintah selanjutnya adalah perintah utama yang di gunakan untuk membuat blog / Website yaitu Jekyll new "nama Blog/ Website" contoh jekyll new "mySite" tunggu beberapa detik sampai muncul New jekyll site installed in <direktori tujuan> untuk step ini blog atau website anda sudah jadi.
